Flow Runtime

Write integration logic in JavaScript

WASL Flow Runtime gives you a JavaScript execution model for request handling, orchestration, HTTP targets, and adapter calls.

Runtime bindings

Flow scripts run with built-in objects such as request, response, flow, http, and adapters.

Simple response example

response.status = 200;
response.setHeader('content-type', 'application/json');
response.body = {
  ok: true,
  runtime: 'wasl-flow-js'
};

Target call example

const result = http.getJson('erp-service', {
  path: '/orders',
  query: { status: 'OPEN' }
});

response.status = 200;
response.body = result;

Adapter call example

const rows = adapters.queryJson('wasl-jdbc', {
  connection: 'default',
  sql: 'SELECT id, name FROM customers'
});

response.status = 200;
response.body = {
  count: rows.length,
  data: rows
};
