Open-Core Model

Adopt for free. Scale commercially.

WASL is built on open-core. Run the high-performance gateways, the Flow runtime, and write plugins using the open SDK for free. Scale to the commercial control plane when you need multi-tenant RBAC, fleet management, and advanced governance.

Free Open Runtime

The data plane is open-source and free to self-host indefinitely. Bring up API, AI, Event, Worker, and MCP gateways on local clusters with no vendor lock-in.

All 5 Runtimes: Serve API, AI, Event, Worker, and MCP traffic Policy Engine: Enforce rate-limits, validation, and standard auth Flow v3 Runtime: Script custom logic using GraalJS bindings Extension SDK: Author custom adapters using the open Java SPI

Extension Strategy

Extend via SDK & Connectors

Backend connectivity is delivered through integration plugins, not hardcoded into the gateway core. Authors write against a stable Java SPI, publish to the built-in marketplace, and dynamically roll out updates.

Hot Reload: Drop a new plugin JAR file without restarting the gateway Cryptographic Signing: Production runtimes reject unsigned or modified extension binaries Telemetry SPI: Custom plugins report logs, metrics, and traces automatically
