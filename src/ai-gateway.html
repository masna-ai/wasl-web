Enterprise LLM Governance

Govern prompts. Control LLM costs.

WASL AI Gateway runs as an isolated thread pool process (`AiGatewayApp`) to proxy LLM requests. It validates, redacts, and caches prompts natively—supporting OpenAI-compatible wire formats with automatic fallback across providers.

The Pipeline

Enterprise Governance Plugin Pipeline

Every LLM request traverses a secure, deterministic plugin chain before routing to OpenAI, Anthropic, Gemini, or self-hosted models.

01 Inbound Auth Authenticate clients via OAuth 2.1 or API Keys mapped from the Developer Portal.
02 Prompt & RAG Inject versioned system prompts and enrich request context using tenant database embeddings.
03 PII Redaction Scan prompts for sensitive data (SSNs, API keys, credentials) and redact them before egress.
04 Semantic Cache Check Redis for embedding-based near-match prompt answers. Save cost and compute instantly.
05 Provider Loop Evaluate weighted load balancing, circuit breaker states, and fallback to alternative LLM providers.
06 Token & Cost Caps Record usage statistics and enforce tenant daily dollar spend ceilings and RPM quotas.
