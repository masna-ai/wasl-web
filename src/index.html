Architecture

Separated Gateway and Studio, designed for production blast-radius control.

Gateway serves data-plane traffic independently of Studio. Studio writes deployment intent to PostgreSQL; Gateway reads the active revision for the tenant and environment. Rollback is a database operation, not file copying.
